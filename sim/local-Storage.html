Local storage is a web storage object which is not sent to the server with each request. When we refresh the page, the values inside the local storage will not be deleted. The types of local storage methods are:

  • localStorage.setItem: Used to set the value and store the key-value pair.
  • localStorage.getItem: Used to get the value from the key.
  • localStorage.removeItem: Used to remove or clear the key-value pair.
  • localStorage.clear: Used to clear all the local storage.
